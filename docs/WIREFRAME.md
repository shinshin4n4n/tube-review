# TubeReview ワイヤーフレーム設計

## 設計方針

### 一覧性重視
- トップページ、ランキング、新着チャンネル一覧では**最小限の情報**のみ表示
- たくさんのチャンネルを素早く見比べられることを優先
- 詳細情報はチャンネル詳細ページで提供

### 情報の段階的開示
- **一覧**: アイコン、名前、評価のみ → 軽量・高速
- **詳細**: 全情報を表示 → リッチ・深い理解

## 一覧カード設計

### 表示要素（最小限）
```
┌─────────────────┐
│  ┌─────┐        │
│  │アイコン│        │
│  └─────┘        │
│                 │
│  チャンネル名    │
│  ★★★★☆ 4.2    │
│  (120件)        │
└─────────────────┘
```

**含まれる情報:**
- チャンネルアイコン（サムネイル）
- チャンネル名
- 平均評価（星5段階）
- レビュー件数

**レスポンシブ対応:**
- デスクトップ: 4列グリッド
- タブレット: 3列グリッド
- モバイル: 2列グリッド

### データ構造（軽量）
```typescript
{
  id: string;              // チャンネルID
  title: string;           // チャンネル名
  thumbnail: string;       // アイコンURL
  averageRating: number;   // 平均評価 (0-5)
  reviewCount: number;     // レビュー件数
}
```

```
│  └────┘  ★★★★☆ 4.2 (120件)        │
│                                      │
│  📅 創設: 2018年5月10日              │
│  🎬 最新投稿: 3日前                  │
│  📺 総動画数: 1,200本                │
│                                      │
│  [チャンネル説明文]                  │
│                                      │
│  ────────────────────────────────  │
│                                      │
│  🏆 代表作                           │
│  ┌────────────┐                    │
│  │サムネイル    │ タイトル            │
│  │             │ 再生数 500万回      │
│  └────────────┘                    │
│                                      │
│  ────────────────────────────────  │
│                                      │
│  💬 レビュー一覧                     │
│  [レビューカード]                    │
│  [レビューカード]                    │
│                                      │
│  ────────────────────────────────  │
│                                      │
│  📊 統計情報（オプション）            │
│  - ジャンル                          │
│  - 動画の平均尺                      │
└──────────────────────────────────────┘
```

### 表示要素（全情報）

#### ヘッダーセクション
- チャンネルバナー画像（YouTube `brandingSettings.image.bannerExternalUrl`）
- チャンネルアイコン
- チャンネル名
- 登録者数
- 平均評価 + レビュー件数

#### 基本情報セクション
- 📅 チャンネル創設日（`snippet.publishedAt`）
- 🎬 最新動画投稿日時（最新動画の `publishedAt`）
- 📺 総動画数（`statistics.videoCount`）
- チャンネル説明文（`snippet.description`）

#### 代表作セクション
- 最も視聴回数が多い動画
  - サムネイル
  - タイトル
  - 再生回数
  - YouTubeへのリンク

#### レビューセクション
- ユーザーレビュー一覧
- レビュー投稿フォーム（ログイン済みユーザー）

#### アクション
- 「見たい」「見ている」「見た」ボタン
- 「リストに追加」ボタン
- 「レビューを書く」ボタン

### データ構造（重量）
```typescript
{
  // 基本情報（一覧と共通）
  id: string;
  title: string;
  thumbnail: string;
  averageRating: number;
  reviewCount: number;
  
  // 詳細ページ専用
  banner: string;              // ヘッダー画像URL
  subscriberCount: number;     // 登録者数
  videoCount: number;          // 総動画数
  publishedAt: string;         // 創設日
  latestVideoDate: string;     // 最新投稿日
  description: string;         // 説明文
  
  // 代表作動画
  topVideo: {
    id: string;
    title: string;
    thumbnail: string;
    viewCount: number;
  };
  
  // レビュー
  reviews: Review[];
}
```

## YouTube Data API 取得戦略

### 一覧表示時（最小限）
- `channels` API: 基本情報のみ
- **1ユニット/チャンネル**
- Supabase にキャッシュ（1日1回更新）

### 詳細ページ遷移時（追加取得）
1. チャンネル詳細情報（バナー、説明文など）: **1ユニット**
2. 最新動画情報: **1ユニット** (uploads playlist)
3. 代表作動画: **初回のみ100ユニット** → Supabase にキャッシュ

### キャッシュ戦略
- チャンネル基本情報: 1日1回更新
- 代表作動画: 1週間1回更新
- 最新投稿日: 1日1回更新

## 主要画面の構成

### 1. トップページ
- ピックアップチャンネル（カルーセル）
- 新着チャンネル（グリッド・一覧カード）
- デイリーランキング（グリッド・一覧カード）

### 2. チャンネル詳細ページ
- 上記「チャンネル詳細ページ設計」参照

### 3. マイリストページ
- タブ切り替え: 「見たい」「見ている」「見た」
- グリッド表示（一覧カード）
- ステータス変更ボタン

### 4. ランキング・新着ページ
- フィルター: カテゴリ、期間
- グリッド表示（一覧カード）
- ページネーション

### 5. テーマ別リスト詳細ページ
- リストタイトル・説明
- 作成者情報
- チャンネル一覧（一覧カード）
- リスト編集機能（作成者のみ）
